Boston Python is a large and vibrant Python user group in the Boston area. We run two events most months. Anyone interested in any aspect of the Python world is welcome!

We use Slack for online discussions.

All of our interactions are covered by our code of conduct.

If you are an employer, you can announce jobs and even sponsor events.
